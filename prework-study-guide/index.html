HTML

  • HTML creates the structure and basic framework of a webpage.
  • HTML uses vatious elements: Text, Image, Header, List, etc.
  • HTML elements use an opening tag "<" and a closing tag ">"
  • The webpage is organized into a "head" element and a "body" element.
  • The head element contains information about the webpage.
  • In the head element, you will start by specifying the document type and language.
  • An HTML Attribute defines additional information about the element.
  • Metadata is data about the webpage that the browser uses to effectively serve the webpage to its users.
  • The body element represents the visible content shown to the user.
  • Use semantic elements when possible, so that the code is easier to read.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
